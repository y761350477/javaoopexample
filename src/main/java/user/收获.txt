快捷键的使用:
构建类:
1. 提取方法内容; -> 方法代码选中状态使用Ctrl+Alt+M;
2. 生成全局变量; -> 变量选中状态使用Ctrl+Alt+F;
3. 方法之间的位置替换; -> Ctrl+Shift+上下方向键;

一个类中存在多个类, 并且要在主方法中实现多个功能的验证, 怎样做才能让每个功能的代码看起来清晰明了?
面临的问题:
1. 主方法中多次对同一个类对象的声明会使对象的命名变得困难和混乱;
2. 每个功能的验证放到一起, 想单个功能验证,需要注释掉不用的功能代码, 这样做会导致注释过多, 特定功能的证变得混乱;
3. 注释看上去显得很不协调;

通过IDEA提供的构建类的快捷键清晰明了的解决了这个难题;
